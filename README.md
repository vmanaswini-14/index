# Parallax Scrolling Web Page

## 📌 Description
This project is a simple Parallax Scrolling Web Page built using HTML and CSS.  
The webpage creates a visual depth effect where background images move slower than the foreground content while scrolling.

## 🚀 Features
- Parallax scrolling effect
- Multiple background sections
- Clean layout using HTML
- Styling with CSS
- Smooth visual scrolling experience

## 🛠️ Technologies Used
- HTML5
- CSS3

## 📂 Project Structure
parallax-project/
│
├── index.html
├── style.css
├── images/
│   ├── bg1.jpg
│   ├── bg2.jpg
│   └── bg3.jpg
└── README.md

## ▶️ How to Run
1. Clone or download the repository
2. Open the `index.html` file in any web browser
3. Scroll the page to see the parallax effect
